## Задача проекта

### _Бизнес задача:_
Определить характеристики, по которым можно выявить клиентов, более склонных к открытию депозита в банке, и за счет этого повысить результативность маркетиноговой компании.

### _Техническая задача как для специалиста в Data Science:_
Построить модель машинного обучения, которая на основе предложенных характеристик клиента будет предсказывать, воспользуется он предложением об открытии депозита или нет.

## Описание проекта
Необходимо проанализировать данные предоставленные банком о последней маркетинговой кампании. Выявить закономерность и найти решающие факторы, повлиявшие на то, что клиент вложил деньги именно в этот банк.


**Основные этапы работы над проектом:**
* Постановка задачи
* Знакомство с данными, обработка пропусков и выбросов
* Разведывательный анализ данных
* Отбор и преобразование признаков
* Решение задачи классификации: логистическая регрессия и решающие деревья
* Решение задачи классификации: ансамбли моделей и построение прогноза
* Вывод

**Что практикуем при выполнении проекта**

Пактикуем и используем все полученные знания на практике. Самостоятельно используем все инструменты для работы с данными и построения модели.

* [Project_4.ipynb](./Project_4.ipynb) - jupiter-ноутбук, содержащий основной код проекта
